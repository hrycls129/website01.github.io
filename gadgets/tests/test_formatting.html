The "del" element in html identifies text that has been deleted. Deleted text usually appears with a line struck through it.

The "ins" element in html identifies text that has been inserted. Inserted text usually appears with an underline.

The "sub" element in html identifies text that is subscripted. For example: H2O.

the "sup" element in html indentifies text that is superscripted. For example: x2.
